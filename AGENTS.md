# Project: CogFun Digital Success Box

## UX & Design Language

- **Vibe:** "Playful Productivity." Clean, rounded corners (Tailwind 'rounded-2xl'), and soft pastel backgrounds.
- **Parent Experience:** "Zero-Friction." The app should allow a parent to log a note in under 3 seconds. Large touch targets are mandatory.
- **Kid Experience:** "Achievement Unlocked." Use badges, simple progress bars, or "card stacks" to represent the Success Box. Visuals > Text.
- **Color Palette:** - Mr. Stop: Calming Blue/Indigo.
  - Mr. Check: Observant Orange/Amber.
  - Mr. Effort: High-Energy Green/Emerald.
- **Characters assets:** generate an emoji for each character. They should be calming and cute.

## Core User Flows

1. **Parent Path:** Open App -> Select Son (Profile A/B) -> Tap "Mr." Agent -> Tap Preset (or Type) -> Done (Visual "Success" confirmation).
2. **Kid Path:** Open App -> Select Profile -> View "Unread" Notes -> Summary view of "Total Successes" by Agent.

## Technical Rules

- Use **Lucide-React** for iconography.
- Implement **Framer Motion** for simple transitions (e.g., notes "sliding" into the box).
- Feature-based folder structure is a must.
